# ClawBrain 项目法典 (GEMINI.md)

1. **AI-Only 闭环**：全代码（源码/测试/配置）由 AI 基于 \`design/\` 提示词生成。Bug 修复必须先改提示词再重构代码。严禁手工干预。
2. **自洽设计溯源**：每份提示词必须是全量功能的完整规格说明（Self-Contained），不依赖历史代码。代码首行必注 \`# Generated from design/[file] v[ver]\`。
3. **分阶段审计存证**：按 Phase 独立留存结果。\`results/\` 须包含关联的 JSON 报告与 Log 审计证据（Input/Expected/Actual 三要素全覆盖）。
4. **全面同步更新**：代码、测试、数据、文档必须四位一体同步变更，严禁挂一漏万。
5. **规范命名体系**：文件名须简明并包含阶段标识（如 \`test_p1_...\`），严禁在文件名中标注版本号（版本号仅限文件内首行）。
