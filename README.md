## 🌟 核心愿景 (Vision)
ClawBrain 是一个仿生学设计的 LLM 代理网关。它作为 OpenClaw 与底层模型之间的“硅基海马体”，通过**三层动力学记忆系统**和**上下文动态提纯**，让你的 AI 能够像人类一样拥有长期记忆、短期注意力和瞬时反应力。

## 🚀 核心特性 (Key Features)
- 🧠 **三代三子记忆系统 (Neural Memory)**：海马体（情节记录）、新皮层（语义泛化）、工作记忆（活跃注意力）。
- ✂️ **上下文高倍提纯 (Context Distillation)**：基于正则的无损压缩，100% 保护代码缩进，平均削减 15% 的冗余 Token。
- 🛡️ **模型准入契约 (TIER Control)**：自动识别模型等级，硬性拦截能力不足的工具调用，确保 Agent 链路不崩溃。
- 🔄 **协议对等路由 (Universal Routing)**：原生支持 Ollama 协议，无缝适配 OpenAI 与未来主流接口。

## 🛠️ 安装指南 (Installation)

```bash
# 克隆并初始化环境
git clone https://github.com/winnerineast/ClawBrain.git
cd ClawBrain && python3 -m venv venv
source venv/bin/activate && pip install -r requirements.txt
```

## ⚙️ 配置挂载 (Mounting to OpenClaw)

修改 `~/.openclaw/openclaw.json`，让 OpenClaw 接入“外挂大脑”：

```json
"ollama": {
  "baseUrl": "http://127.0.0.1:11435",  // 路由至 ClawBrain 神经网关
  "api": "ollama"
}
```
