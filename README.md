## 🌟 核心愿景 (Vision)
ClawBrain 是一个轻量级、高性能的 LLM 代理网关。它作为 OpenClaw 与底层模型之间的“神经节”，通过**模型准入契约**和**上下文动态提纯**，让你的 4090 显卡在处理复杂 Agent 任务时依然保持丝滑秒回。

## 🚀 核心特性 (Key Features)
- 🧠 **模型准入评级 (TIER Control)**：自动识别模型能力，拦截弱智模型的工具调用请求，防止系统崩溃。
- ✂️ **上下文高倍提纯 (Context Optimizer)**：基于正则的无损压缩技术，在 100% 保护代码缩进的前提下，大幅削减 Prompt 体积。
- 🔄 **多协议对等路由 (Protocol Routing)**：完美模拟 Ollama 接口，预留 OpenAI/Claude 扩展位。
- 🛡️ **生产级稳定性**：基于异步连接池与 Lifespan 生命周期管理，支持长时间高并发流式传输。

## 🛠️ 安装指南 (Installation)

### 1. 克隆与环境初始化
```bash
# 克隆仓库
git clone https://github.com/winnerineast/ClawBrain.git
cd ClawBrain

# 创建并激活虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装核心依赖
pip install -r requirements.txt
```

### 2. 配置环境变量
在根目录创建 `.env` 文件：
```env
OLLAMA_BASE_URL=http://127.0.0.1:11434
GATEWAY_PORT=11435
```

---

## ⚙️ 配置挂载 (Configuration)

要让 **OpenClaw** 使用这个外挂大脑，请修改 `~/.openclaw/openclaw.json` 中的 `models` 章节：

```json
"ollama": {
  "baseUrl": "http://127.0.0.1:11435",  // 指向 ClawBrain 的端口
  "api": "ollama",
  "apiKey": "OLLAMA_API_KEY"
}
```
