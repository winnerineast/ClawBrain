# ClawBrain Cognitive Engine: The Self-Paced Memory System

## 1. Architectural Mandate
The memory processing system must be an independent, self-contained subsystem with its own rhythm, fully decoupled from the real-time **Relay Plane**. It does not wait for user requests; it reacts to "Cognitive Pulses" generated by internal and external drivers.

## 2. The Four Drivers of Cognition

### Driver 1: Information Inflow (Stimulus)
*   **Conversational Turns**: Every message processed by the Relay Plane is placed in an ingestion queue.
*   **Vault Watcher**: File system changes in the Obsidian Vault trigger incremental scanning.
*   **Action**: Update L1 Working Memory and L2 Episodic Archive.

### Driver 2: Temporal Decay (The Rhythm of Time)
*   **L1 Charge Leakage**: A background task periodically reduces the "activation" of Working Memory items.
*   **Consolidation Cycles**: After long periods of inactivity, the system triggers "Deep Learning" to move stable patterns from L2 to L3.
*   **Action**: Evict cold L1 items and refine L3 factual summaries during idle hours.

### Driver 3: Volumetric Pressure (The Context Constraint)
*   **Stack Math Monitoring**: When the cumulative character count of all memory layers exceeds defined safety thresholds.
*   **Action**: Force immediate "Lossy Compaction" (distillation) to ensure the AI's context window is never overwhelmed.

### Driver 4: Semantic Boundary (Context Switching)
*   **Topic Detection**: When the `RoomDetector` identifies a shift in the user's intent or project focus.
*   **Action**: Close the current "Semantic Room," finalize its L3 distillation, and prepare a fresh context for the new topic.

## 3. Implementation Logic
*   **MemoryRouter** acts as an **Event Dispatcher**, not a processor.
*   **CognitiveWorker** handles all heavy lifting (Embeddings, LLM calls, Hash calculations) via a non-blocking asyncio task queue.
*   **Isolated Startup**: The system must become `READY` to serve Relay requests within <1 second, with all memory hydration and indexing happening in the background.
